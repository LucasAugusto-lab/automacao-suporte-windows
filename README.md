# automacao-suporte-windows
Script em lote (.bat) interativo com auto-elevação para automação de suporte técnico e gerenciamento de infraestrutura Windows. Centraliza 30 ferramentas essenciais de diagnóstico de rede, manutenção preventiva do sistema operacional, segurança (Firewall/UAC), auditoria de conexões e rotinas de backup automatizado de drivers corporativos.
